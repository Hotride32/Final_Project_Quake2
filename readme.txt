Install:

To Install the Mod use the gamex86.dll and config.cfg in the Final folder in a folder named Final, that is the same as baseq2, in the Quake 2 folder.
Then use the shorcut provided in Final to launch the mod.


Play:

Shift - Dash and Wall Jump.

All weapons changed except Hyperblaster and throwable grenade.

Only have to use once for these effects:
Environment Suit - Half damage.
Rebreather - Dash boots that allow for dash and wall jump.
Quad - Enchanced shots.
